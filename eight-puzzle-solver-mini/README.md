# 8-Puzzle Solver (Mini Full Stack Project)

This is a small student project built using Python, JavaScript, and SQLite.

The application solves the 8-puzzle problem using the A* algorithm with
Manhattan distance heuristic. A simple web interface allows users to enter
a puzzle state and view the solution steps.

## Technologies Used
- Python (Flask)
- JavaScript
- HTML/CSS
- SQLite

## Features
- Solve 8-puzzle using A*
- Display solution steps
- Track number of moves, nodes explored, and execution time
- Save and view last 10 runs from database

## How to Run
pip install -r requirements.txt
python app.py
